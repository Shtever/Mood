Entered or Booked Orders

  • First prior to requesting order be placed on hold PC/PM contacts the Rep (AE, IDSR) with a cc to Rep’s Manager and Internal Sale Support group email (For PSM, AMS, SSM, only notify the Rep) and gives them 30 days before order will be put on hold to be canceled/voided. Notes in the system must be updated and clearly state why the job is being placed on hold and that the Rep has been contacted. Change Forecast date on Installation Master to the end of the next Quarter.

  • PC/PM will do the following on the 31 st day:
    1. Create RMA order if equipment has already been shipped and relate to Installation Master incident, create return label and send to sub/client/tech. Add return label to attachments on Installation Master Incident. If equipment needs to be RTV, follow the RTV process.

      1. If Sales Order is not released yet, PC to make sure Request Date is blank.
      2. NOTE: Do not cancel a Sales Order. Do not Zero out money on a Sales Order.
      3. NOTE: Pay subcontractors if they still need to be paid for any work performed.

    2. Assign Sales Order Owner to Service Delivery Senior Manager[ and changesub type to “cancellation” for all orders needing to be cancelled.
    3. Make status of Installation Parent and Installation Master Child – Hold [*see below if only part of the job is going on hold].
    4. Change Subject: On Hold to be canceled on Installation Parent and Installation Master Child
    5. Close/Completed all other open incidents owned by you.
    6. Assign Installation Parent and Installation Master Child to SD Manager.

  • SD Manager places information onto On Hold spreadsheet and assigns incidents to SD Director after verifying process has been followed to include a return label being sent to retrieve the equipment.

*If only part of the job needs to go on hold for separate sales orders (More than 1 Sales order and one complete order needs to go on hold):

  1. Create New Installation Master Incident and mark status as Hold. Relate to Parent Installation Incident.
  2. Change Subject: On Hold to be canceled for New Installation Master Incident
  3. Installation Parent will continue through the regular RTB/Billing process. (DO NOT change the status of the Installation Parent to Hold)
  4. Relate Sales Order and/or RMA order if applicable that needs to be placed on hold to the new Installation Master. Remove the relationship to the old Installation Master.
  5. Assign the new Installation Master Child to SD Senior Manager.
